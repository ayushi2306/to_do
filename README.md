# 📝 Console-Based To-Do List (Python)

## 📌 Overview

This project is a beginner-friendly command-line To-Do List application built using core Python concepts. It allows users to manage their daily tasks directly from the terminal through a simple menu-driven interface.

The purpose of this project was to strengthen my understanding of Python fundamentals by applying them to a real-world application.

---

## ✨ Features

* Add new tasks
* View all added tasks
* Delete existing tasks
* Input validation for invalid menu choices
* Numbered task display

---

## 🛠️ Technologies Used

* Python 3

---

## 📚 Python Concepts Practiced

* Variables
* Lists
* User Input (`input()`)
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`while`, `for`)
* List Methods (`append()`, `pop()`)
* `enumerate()`
* Basic Program Flow
